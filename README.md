# Android_video_app
### 开发环境
● 界面设计器： Android Studio
● 软件编译器： Idea
● Web服务器： tomcat9
● Java 开发包： JDK 1.8
● 数据库： Oracle
### (1) 系统前台功能：
未登陆用户： 1. 用户注册  2.浏览主页  3.查看电影 4.搜索电影 5. 查看评论

已登陆用户： 1. 用户登陆 2. 资料修改 3. 修改密码 4. 浏览主页 5. 观看电影 6. 电影评论 7. 电影评分 8. 搜索电影 9. 收藏电影 10. 查看评论 11. 用户注销（退出）

### (2) 系统后台功能：
管理员：
1. 用户信息管理：会员信息浏览、修改、删除；
2. 视频管理：上传视频，浏览视频，修改视频，删除视频；
3. 安全退出。
